Factsheet

Game
ZpellCatz

Status
In development, closed beta

Developer
SimaGames (Florian Siemer)
Solo developer from Europe (Germany)
Founded in 2014

Genre
Action RPG

Platforms
Steam Deck, Windows, Linux

Pitch

ZpellCatz is a magical Action RPG set in a colorful fantasy world (Zelda 1 meets Diablo 2 - with cats).

Description

You are a cat with magic powers, stranded on a cursed island. Spiteful dogs are lurking in every corner and a fateful evil is about to emerge from the darkness. ZpellCatz is a magical Action RPG set in a colorful fantasy world.

Timeline

  • Late 2018: Project start (in its current form)
  • Late 2021: Announcement with trailer, Steam page, etc
  • Late 2021: Prologue released
  • Mid 2023: Start for the private beta (full game)
  • Early to mid 2024: Planned release for the full game
